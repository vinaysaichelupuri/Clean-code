# Clean code

### It is a mindset of writing code


## Pre-requisite
Pre-requisite: Node 22.11.0 and npm > 8

## Installations
1. Clone the repository
   ```shell
   https://github.com/vinaysaichelupuri/Clean-code.git
   ```
2. Install dependencies
   ```shell
   npm install
   ```
3. Run Tests
   ```shell
   npm run test
   ```
